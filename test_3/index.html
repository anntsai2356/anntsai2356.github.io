一台電腦

熟悉 Mac 操作

  1. 快捷鍵（請參考 Apple 官方說明）
  2. 手勢（請參考 Apple 官方說明） 亦可在系統偏好設定

個人化環境設定

在「系統偏好設定」中設定一開始會用到的幾個項目

Linux 基本概念

學習資源：鳥哥的 Linux 私房菜

請掌握 第 6 章（Linux 檔案與目錄管理） 和 第 9 章（vim 程式編輯器） 的內容，有空的話至少看到基礎文件的第 13 章節，此目的是為了確保你清楚知道自己對電腦做了什麼事情，以免憾事發生。

安裝 MacOS 的套件管理工具：Homebrew

  1. 按照官網指示安裝，執行以下指令

    1

    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  2. 看到這個後，輸入密碼（電腦的密碼）

  3. 重啟 iTerm2，並輸入以下指令確認 homebrew 有在運作

    1

    brew doctor

:::danger 題外話： 若未來有需要更新，在執行 brew update 之前，請確認目前作業系統版本是否符合homebrew 最新版本的最低要求。（homebrew 僅支援最近三個 MacOS 版本）

若低於最低要求，請勿貿然進行更新，建議先進行環境評估。 :::

準備好用的 Terminal 環境（非必要）

安裝取代預設 Terminal 的工具：iTerm2

  1. 官網下載並解壓縮開啟
  2. 將程式移到 應用程式（application） 資料夾（系統會跳出視窗詢問）
  3. 之後都改用 iTerm2 執行

安裝取 BASH 的工具：ZSH （Z shell）

  1. 執行以下指令
    1

    brew install zsh zsh-completions
  2. 設為預設 shell
    1
2
3

    sudo sh -c "echo $(which zsh) >> /etc/shells"

chsh -s $(which zsh)
  3. 重啟 iTerm2，並輸入以下指令確認是否安裝成功
    1

    echo $SHELL

安裝框架用以管理外掛及主題：oh-my-zsh

透過 curl 安裝

1

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

或 透過 wget 安裝

1

sh -c "$(wget -O- https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

修改主題

  1. 選擇主題 選擇喜歡的主題（查看主題列表）
  2. 進入 ~/.zshrc
    1

    vim ~/.zshrc
    修改 ZSH_THEME=”robbyrussell” ，改成欲改的主題名稱
    1

    ZSH_THEME="主題的名稱"
    也可以選擇隨機顯示
    1

    ZSH_THEME="random" #隨機主題
    修改完離開，執行
    1

    source ~/.zshrc

:::info 【補充】Vim 的使用方法

進入 ~/.zshrc

1

vim ~/.zshrc

:::

安裝資料庫：MySQL

  1. 查看可安裝的版本

    1

    brew search mysql

  2. 安裝公司使用的 5.6 版本

    1

    brew install mysql@5.6

:::warning 不小心使用 brew install mysql 安裝到最新的話，可以用 brew uninstall mysql 解除安裝，再安裝正確的版本。要注意其他資料夾中有關 MySQL 的檔案都清乾淨，可自行搜尋「Mac/解除安裝/MySQL」。 :::

  1. 啟動 MySQL

    1

    mysql.server start

    若要終止運作，可輸入以下指令

    1

    mysql.server stop

  2. 修改環境變量，並確認已加入此路徑到 ~/.zhsrc 檔案最下面

    1

    export PATH="/usr/local/opt/mysql@5.6/bin:$PATH"
